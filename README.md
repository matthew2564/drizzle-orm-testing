# Typescript ORM DB testing

## Pre-requisites
- [npm](https://www.npmjs.com/) or [bun](https://bun.sh/)

## Getting started
1) Navigate into desired package within `packages/`
2) `npm`/`bun` install
3) `npm`/`bun` run test

###### Bun can run TS files directly without the need for `ts-node`. e.g. `bun src/index.ts`
